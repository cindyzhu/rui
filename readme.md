# rui   
> 简介     

rui为css前端框架 目前仅用于原子贷业务线;使用sass作为语法糖，用glup

> 目录介绍   

|--sass  为源码目录       
|----lib  为类库源文件         
|----usage 为用例代码（如何使用参考这个目录）       
|-css 为编译后的代码  
|-demo 为用例展示页
|-font 为字体文件  


> 如何查看  

点开demo里面的index.html 就能查看现在支持的css组件

> start
```
$ npm install ruicss --save-dev
```
